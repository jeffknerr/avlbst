# avlbst
[Adelson-Velsky and Landis](https://en.wikipedia.org/wiki/AVL_tree) (AVL) Binary Search Tree (BST) in python

## see/try example code:
    python3 example.py
    python3 avlexample.py

## run the tests
    python3 tests.py
